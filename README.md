# GTA 1 HTML Prototype (Mobile Controls)

Мини-прототип top-down игры в духе GTA 1 на чистых HTML/CSS/JS с мобильным управлением.

## Что уже есть

- Бесконечный город-сетка с камерой, следующей за машиной.
- Управление машиной:
  - Клавиатура: `WASD` / стрелки.
  - Сенсор: левый джойстик + кнопки `Газ` и `Тормоз`.
- HUD со скоростью и координатами.

## Запуск

Открой `index.html` в браузере или подними простой статический сервер:

```bash
python3 -m http.server 8080
```

После этого перейди на `http://localhost:8080`.
