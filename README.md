# GTA 1 HTML Prototype (Mobile Controls)

Мини-прототип top-down игры в духе GTA 1 на чистых HTML/CSS/JS с мобильным управлением.

## Играть сразу по ссылке (GitHub Pages)

Я добавил автодеплой через GitHub Actions (`.github/workflows/pages.yml`).

После пуша в ветку `work` включите Pages в репозитории:

1. GitHub → **Settings** → **Pages**.
2. В `Source` выберите **GitHub Actions**.
3. Запустится workflow деплоя, после него игра будет доступна по прямой ссылке:
   - `https://<your-github-username>.github.io/GTA1-HTML/`

Это и есть «прямая ссылка», которую можно сразу открыть на телефоне и играть.


## Custom domain

Для вашего кастомного домена уже добавлен файл `CNAME` со значением `forgot`.

Что сделать в GitHub:

1. `Settings` → `Pages`.
2. Убедиться, что source = **GitHub Actions**.
3. В разделе custom domain должен отобразиться `forgot` (из файла `CNAME`).
4. Настроить DNS у регистратора:
   - для apex-домена: A/AAAA записи на GitHub Pages IP,
   - для поддомена: CNAME на `<your-github-username>.github.io`.
5. Дождаться проверки DNS и выпуска HTTPS-сертификата.

После этого игра будет открываться напрямую через ваш домен `forgot`.

## Локальный запуск (прямо сейчас)

Открой `index.html` в браузере или подними статический сервер:

```bash
python3 -m http.server 8080
```

Далее открой:

- `http://localhost:8080`


> Если раньше кнопки не реагировали на телефоне, обновите страницу (hard refresh).
> В текущей версии удержание `Газ/Тормоз` работает стабильно и в Safari на iPhone.
